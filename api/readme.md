**OWASP iGNITA API**
----
*POST CODE*

* **URL**

  <http://localhost:3000/upload/code>

* **Method:**
  
  `POST`
  
*  **URL Params**

   No Params 

   **Required:**
 
   code field of type file

   **Optional:**
 
   No Optional fields

* **Data Params**

`code` param of `File` Type 

* **Success Response:**
  

  * **Code:** 200 <br />
    **Content:** `[
    File is uploaded
]`
 

* **Sample Call:**
$ curl -F ‘data=@path/to/local/file’ UPLOAD_ADDRESS
